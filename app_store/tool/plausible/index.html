什么是 AFFiNE

  • Plausible 是一款轻量级、开源且注重隐私的流量分析工具，无需使用 cookies，操作简洁高效。它提供关键的流量指标，界面直观，无冗余信息，对网站性能影响极小。Plausible 支持自托管和 SaaS模式，非常适合需要基础流量统计且重视用户隐私的个人和企业使用。

功能列表

  • 快速且易于集成：设置简单，可与 WordPress、Ghost、Squarespace、Wix 和 Weebly 等流行工具集成。支持单页应用程序和基于哈希的路由。
  • 强大而灵活的 API：我们的 API 提供了一种以编程方式检索您的统计数据、将您的指标发送到您想要的任何位置并以您想要的任何形状呈现它们的方法。
  • 电子邮件和 Slack 报告：通过每周和/或每月报告密切关注您的流量。获取流量高峰通知，这样您就不会错过 Hacker News。报告可以发送给多个收件人。
  • 将脚本作为第一方连接： AFFiNE有一个活跃的社区，提供了丰富的插件和扩展，可以根据你的需求定制软件的功能。例如，代码高亮、任务管理、日历集成等插件应有尽有。
  • 拥有并控制您的数据： 您的站点数据绝不会与任何第三方共享或出售。它绝不会被货币化、挖掘和收集，用于个人和行为趋势。
  • 与 Search Console 集成：与 Google Search Console 集成，即可直接在 Plausible 仪表板中查看人们用来在 Google 搜索结果中找到您的网站的搜索查询。
